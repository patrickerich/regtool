# Register Tool

A register generation tool for hardware designs that generates:
- SystemVerilog RTL
- UVM register models
- C headers
- Documentation (Markdown/HTML)

## Installation
```bash
pip install git+https://github.com/patrickerich/regtool.git
```

## Status
This project is in an early stage of development.
 

## Usage
```bash
regtool --help
```
